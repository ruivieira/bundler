# bundler

Web assets bundler. Pre-alpha. API will break.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  bundler:
    github: ruivieira/bundler
```

### Dependencies

Minifying Javascript will depend on `uglify-js`. Install it with

```
npm install -g uglify-js
```

## Usage


```crystal
require "bundler"
```
